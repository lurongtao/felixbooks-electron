构建 Vue 项目基本结构

准备 Header, Main, 和 Modal 三个组件。

0、reset.css 样式

编写 reset.css 样式：

/* /src/assets/styles/reset.css */

html, body {
  height: 100%;
}

body {
  font: caption;
  margin: 0;
  display: flex;
  flex-flow: column;
}

1、Header 组件

在 components 文件夹下创建 Header.vue 组件： /src/components/Header.vue

<template>
  <header>
    <button id="show-modal">+</button>
    <input type="text" id="search" placeholder="Search">
  </header>
</template>

<script>
export default {

}
</script>

<style lang='stylus' scoped>
button {
  background: dodgerblue;
  color: white;
  border-radius: 5px;
  border: none;
  font-size: 20px;
  outline: none;
}

input {
  font-size: 20px;
  border-radius: 5px;
  border: 1px solid silver;
  padding: 0 10px;
}

input::placeholder {
  color: lightgray;
}

header {
  background: lightgray;
  display: flex;
  padding: 10px;
  font-weight: bold;
  border-bottom: 1px solid silver;
  box-shadow: 0px 10px 10px rgba(0,0,0,0.1);
}

#show-modal {
  padding: 0px 12px 5px;
  margin-right: 10px;
  font-size: 30px;
}

#search {
  flex-grow: 1;
}
</style>
